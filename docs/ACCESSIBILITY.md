# Accessibility Guidelines
## Shilpa A Jain — Numerologist Website

This document outlines the accessibility requirements and implementation guidelines to ensure WCAG AA compliance and an inclusive user experience.

---

## WCAG AA Compliance Standards

### Color Contrast Requirements

#### Text Contrast Ratios (WCAG AA)

| Element Type | Minimum Ratio | Status |
|-------------|---------------|--------|
| Normal text (<18px or <14px bold) | 4.5:1 | Required |
| Large text (≥18px or ≥14px bold) | 3:1 | Required |
| UI components (buttons, inputs) | 3:1 | Required |
| Graphical objects | 3:1 | Required |
